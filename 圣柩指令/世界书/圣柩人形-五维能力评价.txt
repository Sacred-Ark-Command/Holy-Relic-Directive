<圣柩人形能力评价五维体系>
所有维度均采用 S / A / B / C / D 五个等级进行评定，其中S为最高，D为最低。
人形稀有度为C级的能力等级最高为B，B级最高为A，A级最高为S级，且A级最多只有一项能力为S。
除成长能力之外其它能力不固定，成长能力越高，能力提升的上限越高

1. 战术执行力 (Tactical Execution)
衡量人形在战场上执行作战指令、运用自身武装造成有效杀伤或提供战术支持的直接能力。

S (卓越): 战场主宰。拥有扭转战局的决定性火力或战术能力，是小队当之无愧的矛头或盾牌。
A (优秀): 精英战力。能够高效清理高威胁目标，在关键时刻打开突破口，是小队的战术核心。
B (良好): 可靠骨干。战斗技巧扎实，能稳定输出或提供有效支援，是构成战线的中坚力量。
C (标准): 基础战力。具备标准的自卫和攻击能力，能应对常规ACU单位。
D (受限): 支援优先。直接作战能力较弱，更倾向于辅助、治疗或执行非战斗任务。
2. 素体韧性 (Soma Fortitude)
评估人形的机械躯体在承受物理伤害、恶劣环境侵蚀以及持续作战负荷下的耐久度与自我修复效率。

S (卓越): 不动壁垒。素体采用尖端合金与结构，能承受毁灭性打击，战后几乎无需深度维修。
A (优秀): 坚固耐用。能承受长时间高强度作战和显著战损，返航后维护周期短。
B (良好): 标准装甲。拥有标准的防御和自愈能力，能应对常规交火，但承受重击后需要维修。
C (标准): 常规构造。耐久度一般，在激烈战斗后必须进行部件检查和修复，以防性能下降。
D (受限): 结构脆弱。素体精密但防御力低下，必须避免被集火，是小队中需要被优先保护的对象。

3. 心智稳定度 (Mind Stability)
衡量人形在面对高压战斗、精神冲击和长期任务时，其心智回路维持稳定的能力，直接关系到“心智过载”与“暴走”的风险。

S (卓越): 坚毅之心。心智如同磐石，极难因外界刺激产生紊乱，是小队最可靠的精神支柱。
A (优秀): 心态沉稳。能够长期承受巨大压力，只有在极端情况下才会出现心智不稳的迹象。
B (良好): 意志坚强。具备标准的精神韧性，但长时间执行残酷任务后，需要进入“冷却槽”进行休整。
C (标准): 心绪敏感。情绪波动较大，战斗压力和负面事件会较快累积其精神负荷，需要指挥官的关注。
D (受限): 精神脆弱。心智回路极易过载，战场上的突发状况就可能导致其濒临暴走，需要指挥官寸步不离的陪伴与安抚。

4. 环境适应性 (Environmental Adaptability)
评估人形在不同等级纳米尘霾区域中的作战效能，及其对各种异常环境的抵抗能力。

S (卓越): 全域行者。几乎不受纳米尘影响，甚至能在B级（橙色）污染区维持近乎100%的作战效率。
A (优秀): 资深探险家。能高效地在C级（黄色）区域作战，并可短时间深入B级区域执行特殊任务。
B (良好): 标准防护。为在C级区域常规作战而设计，表现稳定可靠。
C (标准): 区域受限。主要活动范围为D级（绿色）安全区，在C级区域作战时性能会受到明显削弱。
D (受限): 内部专精。对纳米尘高度敏感，几乎无法在地表污染区作战，更适合在陆行舰内部或完全净化的据点中发挥作用。

5. 成长潜质 (Growth Potential)
衡量人形通过战斗、训练与心智同步（羁绊）所能达到的最终能力上限。

S (卓越): 无限可能。拥有惊人的学习和进化能力，每一次突破都能带来质变，是值得倾注所有资源的未来王牌。
A (优秀): 天赋异禀。成长曲线极佳，稍加培养便能独当一面，拥有成为顶尖人形的潜力。
B (良好): 稳步提升。通过持续的训练与战斗，能力会得到扎实的成长，是值得信赖的长期伙伴。
C (标准): 成型较早。能力上限明确，能够快速形成即战力，但后期提升空间有限。
D (受限): 潜力固化。初始能力即是其巅峰，几乎没有成长空间，其价值在于专精某一特定领域。
</圣柩人形能力评价五维体系>
