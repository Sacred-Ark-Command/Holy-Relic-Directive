# Holy-Relic-Directive

《圣柩指令》SillyTavern 角色卡项目。

主要入口：

- `圣柩指令/圣柩指令.yaml`: 角色卡主索引，注册世界书、正则与扩展字段。
- `圣柩指令/头像.png`: 角色卡头像。
- `圣柩指令/第一条消息/`: 开场消息文本。
- `圣柩指令/正则/`: 正则显示模板。
- `圣柩指令/世界书/`: 世界书正文，按当前功能目录分类。
- `导出/圣柩指令.png`: 当前导出的角色卡图片。

世界书主要目录：

- `世界书/COT/`
- `世界书/变量/`
- `世界书/系统/`
- `世界书/世界设定/`
- `世界书/圣柩人形/`
- `世界书/地区/`
- `世界书/势力与组织/`
- `世界书/剧情与事件/`
- `世界书/玩法(Plan)/`
- `世界书/游戏机制/`
- `世界书/游戏机制/陆行舰/`
- `世界书/游戏机制/任务与资源/`

维护规则和目录说明见 `AGENTS.md`。
